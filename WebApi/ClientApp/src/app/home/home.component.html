Code Challenge!

Welcome to my solution for a code challenge, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

Responsabilities:

  • Currency Rates. Get Currency rates from http://www.bancoprovincia.com.ar/Principal/Dolar.
  • Buy Currency. Based on limits configuration user will be able to buy a certain exchange from ARS to USD or BRL.
